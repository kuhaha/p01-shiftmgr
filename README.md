各機能操作マニュアル
・ログイン機能
　ログイン画面ではユーザ名とパスワードを入力することにより、トップメニューへ移動することができる。ユーザ名もしくはパスワードを誤った場合、新しく正しいものを入力する。

・従業員トップページ機能
　従業員専用のトップページ画面になっており、シフト希望の登録と従業員のシフト希望状況を確認することができる。

・シフト希望提出機能
　従業員は1か月毎のシフトを提出する。営業予定は管理者でスケジュール調整をする。追加時給に関しても管理者側で、忙しくなる予定がある場合や人が不足しているときなどに措置をとる。希望したいシフトを選びたい場合は、A・B・C・Dの中から入れるものを選択する。入れない場合はOFFボタンを選択する。もし、間違えて入力してしまった場合でも変更は可能である。（A:6:00〜12:00、B:12:00〜17:00、C:17:00〜22:00、D:22:00〜6:00）

・管理者トップページ機能
　管理者限定のトップメニュー画面となっており、営業スケジュールの登録とシフトの決定を行うことができる。以降は従業員と同様である。

・営業スケジュール登録機能
　管理者権限のみでの機能となっており、営業予定・イベント・追加時給の内容を登録・変更することができる。営業予定と追加時給の登録内容は、従業員のシフト希望登録画面でも確認することができる。追加時給に関しては、営業状況に考慮したシステムとして従業員に入っていただきたい時に追加時給をとることで、人手不足も解消でき、会社にとってシフト管理もしやすくなる。また、追加時給に登録できる値は0~5円間までである。

・シフト決定機能
　管理者のみでの機能となっており、シフトを決定させたい従業員のチェックボックスにチェックすることでシフトの決定を行うことができ、一度チェックしたボタンは次回もチェック済みである。また、シフトを決定した場合はトップメニュー画面に“決”の文字が表示される。

・人事トップページ機能
　人事限定のトップメニュー画面となっており、基本時給の設定と個人加給の設定を行うことができる。以降は従業員と同様である。

・基本時給設定機能
　年度ごとの売り上げは変更不可能だが、基本時給の変更は設定したい値をテキストボックスに入力し、設定/再設定をクリックすると変更可能である。2014年と2015年の売上げに関しては確定したものであるが、2016年に関しては予測の売上げとなっている。

・個人加給設定機能
　従業員別の時給設定機能となっている。勤続日数と勤務態度評価に関しては変更は不可能だが個人の加給は変更可能である。テキストボックスの中の値を変更して、設定/再設定ボタンをクリックすることで加給の変更が可能である。
